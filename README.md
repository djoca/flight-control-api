# Flight Control API

This is a simple Flight Control Rest API made with Spring Boot. 

## Getting started

1. Make sure you have Git, Java 1.8 and Maven 3.5 installed.
2. Clone this repository locally.
3. Execute `mvn spring-boot:run` to start the server.
